## Short Description

## Full Description

**Generate OpenAPI/Swagger documentation from any web app — automatically.**

FastAPI Doc Gen is a Chrome extension that sits in your toolbar and automatically detects API calls on any web page you visit. It captures endpoints from:

- **Fetch API calls** — intercepts all `fetch()` requests
- **XHR requests** — intercepts all `XMLHttpRequest` calls
- **API documentation links** — finds `/api/`, `/docs`, Swagger/OpenAPI links
- **Embedded specs** — detects OpenAPI/Swagger JSON in `<script>` tags
- **Axios instances** — detects axios base URLs

Then it generates a complete **OpenAPI 3.0.3 specification** that you can:

- **Copy to clipboard** — paste directly into your IDE or docs
- **Download as JSON** — save as `openapi.json` for use in Postman, Swagger UI, or code generators
- **Use to generate client SDKs** — feed into OpenAPI Generator for 50+ languages

### How It Works

1. Click the extension icon on any web page
2. Click "Scan Page for APIs" — the extension captures all API endpoints
3. Click "Capture Endpoints" to save them
4. Click "Generate OpenAPI" to create a complete OpenAPI 3.0.3 spec
5. Copy or download the spec
